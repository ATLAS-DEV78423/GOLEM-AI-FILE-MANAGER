# FAQ

## Is GOLEM cloud-only?

No. It can run in heuristic mode without any API key.

## Does it work with Obsidian?

Yes. GOLEM writes notes into the configured vault.

## Does it overwrite my notes?

It skips notes that are marked as user edited.

## Does it support OCR?

Not yet.

## Does it support multiple folders and vaults?

Not fully yet.

## Can I undo changes?

Yes. The app keeps an undo log for the latest organization action.
